Magic The Gathering Manager est destiné aux passionnés des cartes Magic l'Assemblée ayant beaucoup de cartes et désirant gérer leur collection. Il permet de saisir toutes les cartes (en anglais ou en français) dans des groupes différents. La base de données fournit avec le logiciel contient toutes les références jusqu'à l'édition Les Cicatrices de Mirrodin, mais un module permet de compléter la base de données avec les éditions futures, via le site de Wizard of the Coast. Les cartes saisies peuvent ensuite être recherchées, affichées et triées selon des critères entièrement paramétrables. Il est possible d'obtenir une estimation du prix de ses cartes, ceux-ci pouvant être mis à jour via le site de MagicCorportation. D'autres statistiques très intéressantes sont également disponibles, permettant d'optimiser le choix des cartes de ses decks. Un module permet également l'achat de cartes à d'autres joueurs à prix optimal via le site de MagicVille. Un logiciel à tester d'urgence pour les fans de Magic !
